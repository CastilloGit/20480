You are developing an e-commerce website. Users place products into a shopping cart. You need to ensure that the shopping cart data is available between browser sessions. Which object should you use to store the shopping cart data? A. clientStorage B. localStorage C. applicationStorage D. sessionStorage <b>Respuesta correcta: localStorage</b> Explicación: la información que para un servidor web de un site e-commerce ocuparia demasiado con cientos de usuarios, es almacenada de manera local ganando además en seguridad y en volumen de información. Anteriormente esta información se captaba a través de las cookies. La propiedad de sólo lectura localStorage te permite acceder al objeto local Storage; los datos persisten almacenados entre de las diferentes sesiones de navegación. localStorage es similar a sessionStorage. La única diferencia es que, mientras los datos almacenados en localStorage no tienen fecha de expiración, los datos almacenados en sessionStorage son eliminados cuando finaliza la sesion de navegación - lo cual ocurre cuando se cierra la página.
